# Шашки Онлайн ♟️

Полноценная мультиплеерная игра в русские шашки с онлайн-соперником, очередью Firebase, красивой анимацией и настраиваемым внешним видом. Проект создан на React Native (Expo).

## ✨ Возможности
- Игра с реальным соперником через интернет (Firebase Realtime Database)
- Очередь ожидания и автоматическое создание комнат
- Полные правила русских шашек:
  - обычные шашки ходят вперёд на одну клетку, бьют вперёд и назад
  - обязательное взятие
  - цепочка взятий
  - дамки (ходят на любое расстояние, бьют через фигуру)
- Адаптивный интерфейс – ваши фишки всегда внизу
- Визуальные подсказки: подсветка доступных ходов, пульсация фигур, которые могут взять
- Кастомизация цветов доски, своих фишек, фишек противника и короны дамки
- Счётчик съеденных фишек

## 📋 Системные требования
- Операционная система: Windows, macOS или Linux
- [Node.js](https://nodejs.org/) (версия 16 или выше) – среда выполнения JavaScript
- npm (устанавливается вместе с Node.js) или yarn
- [Android Studio](https://developer.android.com/studio) – для запуска эмулятора Android (не обязательно, если будете тестировать на реальном телефоне через Expo Go)
- Аккаунт [Firebase](https://firebase.google.com/) (бесплатный)

## 🚀 Пошаговая инструкция по установке и запуску на новом компьютере

### 1. Установка Node.js и npm
1. Перейдите на [официальный сайт Node.js](https://nodejs.org/).
2. Скачайте **LTS-версию** (рекомендуется) и запустите установщик.
3. В процессе установки убедитесь, что опция **«Add to PATH»** отмечена.
4. После завершения откройте **командную строку (cmd)** или **PowerShell** и проверьте:
   ```bash
   node --version
   npm --version
